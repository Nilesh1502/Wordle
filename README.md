# Wordle

A command line wordle game using python.<br />


### Requirements:<br />
Python version-3<br />
Install colorama: **pip install colorama** <br />
<br />

### Run the code:<br />
Make sure to fulfill the requirements.<br />
In terminal: **py game.py** / **python game.py** to run the application.<br />
